1. Inleiding en systeemoverzicht

De Finally Card is een op maat gemaakt Home Assistant dashboard voor schepen met een Victron Energy systeem. Het bestaat uit drie componenten:

  • finally-skycard.js — volledig scherm kiosk dashboard voor een vaste display (bijv. Dell in de stuurhut)
  • finally-skycard-mobile.js — mobiel dashboard voor de HA app op telefoon of tablet
  • finally-wizard.js — installatiewizard die het systeem automatisch configureert

Het systeem monitort en beheert:

  • Accu SOC, spanning en stroom (via SmartShunt)
  • Zonne-energie productie (via SmartSolar MPPT)
  • Walstroom aansluiting (via Quattro/Multiplus + Shelly schakelaar)
  • JK BMS cel spanning bewaking
  • Klimaat, verlichting en systeem status
De Finally Card werkt uitsluitend met Victron apparatuur aangesloten op een Cerbo GX of Venus GX. Zonder GX apparaat is de integratie niet mogelijk.

2. Benodigde hardware

Minimale configuratie

  • Mini-PC voor Home Assistant (bijv. HP EliteDesk 705 G3 USFF, Beelink Mini S12 Pro, of vergelijkbaar)
  • Minimaal 4GB RAM, 32GB opslag (SSD aanbevolen)
  • Netwerkkabel of WiFi op hetzelfde netwerk als de Cerbo GX
  • Shelly Plug S of Shelly Plus Plug S voor walstroom schakelaar

3. Home Assistant installeren (HAOS)

Voorbereiding

  1. Download Balena Etcher: https://etcher.balena.io
  2. Download het HAOS image voor x86-64:
    https://github.com/home-assistant/operating-system/releases
  3. Kies: haos_generic-x86-64-[versie].img.xz
  4. Schrijf het image naar een USB stick (minimaal 8GB) met Balena Etcher

BIOS instellen

  1. Start de mini-PC op en druk op F2 / Del / F10 om in de BIOS te komen (verschilt per merk)
  2. Zet UEFI boot mode AAN
  3. Zet Secure Boot UIT
  4. Zet USB als eerste boot optie
  5. Sla op en herstart
HP EliteDesk tip: druk F10 voor BIOS setup. UEFI staat standaard aan. Secure Boot staat onder Security.

Installatie

  1. Steek de USB stick in de mini-PC
  2. HAOS installeert zichzelf automatisch op de interne SSD — dit duurt 5-10 minuten
  3. Wacht tot het scherm "Use browser to configure" toont
  4. Ga op een andere PC/tablet naar http://homeassistant.local:8123 of het IP adres dat je in de router ziet
  5. Volg de onboarding wizard en maak een admin account aan
  6. Wachtwoord: gebruik een uniek wachtwoord per installatie, bijv. Finally-[bootnaam]-[jaar]
Bewaar het wachtwoord per klant in een wachtwoordmanager. Gebruik nooit hetzelfde wachtwoord voor meerdere installaties.

4. Victron integratie instellen

  1. Ga in HA naar Instellingen → Apparaten & Diensten
  2. Klik op + Integratie toevoegen
  3. Zoek op "Victron GX" en selecteer de integratie
  4. HA detecteert de Cerbo GX automatisch als deze op hetzelfde netwerk zit
  5. Selecteer de gevonden Cerbo en klik Voltooien
Als de Cerbo niet automatisch gevonden wordt: controleer of "MQTT on LAN" aan staat via de Remote Console → Services.

Apparaten na integratie

  • GX Device
  • SmartShunt (accubewaking)
  • Quattro of Multiplus (omvormer/lader)
  • SmartSolar MPPT (zonnelader)
  • Generator start/stop (indien aanwezig)

5. Shelly schakelaar instellen

  1. Verbind de Shelly met het WiFi netwerk via de Shelly app
  2. Zorg dat de Shelly op hetzelfde netwerk zit als de HA machine
  3. Ga in HA naar Instellingen → Apparaten & Diensten
  4. De Shelly integratie wordt automatisch gevonden — klik Configureren
  5. Noteer de entity ID van de schakelaar (bijv. switch.shellyplus1_78ee4cc39480)
Gebruik een Shelly Plug S of Shelly Plus Plug S. Andere merken (bijv. Calex, TP-Link) werken mogelijk alleen via cloud en zijn niet betrouwbaar voor automatisering.

6. Finally Card wizard installeren

Gebruik bij klantinstallaties altijd de -customer bestanden (bijv. finally-wizard-customer.js), nooit de bestanden zonder die toevoeging. Die laatste zijn alleen voor de eigen MS Finally installatie en bevatten boot-specifieke elementen.

Bestanden uploaden via Samba

  1. Installeer de Samba Share addon:
    Instellingen → Add-ons → Add-on store → zoek "Samba share" → Installeren → Starten
  2. Open Windows Verkenner en ga naar: \\[IP-adres]\config
  3. Maak een map www aan als die nog niet bestaat
  4. Kopieer de volgende bestanden naar de www map:
    • finally-wizard-customer.js
    • finally-skycard-mobile-customer.js
    • finally-skycard-customer.js (alleen bij kiosk installatie met vast scherm)
  5. Maak een map www/finally-card/ aan en kopieer hierin:
    • Alle 13 weerachtergronden (clear-day.png, cloudy-night.png, etc.)
    • De voorbewerkte boot.png van de klant (zie sectie 6b)
  6. Herstart HA: Instellingen → Systeem → Herstarten

Resource toevoegen in HA

  1. Ga naar Instellingen → Dashboards → drie puntjes (⋮) → Resources
  2. Klik + en voeg toe:
/local/finally-wizard-customer.js          →  JavaScript module
/local/finally-skycard-mobile-customer.js  →  JavaScript module
/local/finally-skycard-customer.js         →  JavaScript module (kiosk)

Wizard dashboard aanmaken

  1. Ga naar Instellingen → Dashboards → + Dashboard toevoegen
  2. Geef het de naam "wizard"
  3. Open het dashboard → bewerk (potlood rechts boven) → + Card toevoegen → Handmatig
  4. Voer in:
type: custom:finally-wizard-customer
  1. Sla op — de wizard verschijnt nu in het dashboard

Activatiecode

De wizard vraagt bij eerste gebruik om een licentiecode. Deze genereer je via de codegenerator (finally-codegenerator.html) of vraag deze op bij Erik. Formaat: FC-[jaar]-[naam]-[checksum], bijv. FC-2026-TWIN-8511.

Wizard doorlopen

  1. Stap 0: Activatiecode invoeren
  2. Stap 1: Voer de naam van het schip in
  3. Stap 2: Selecteer welke apparatuur aanwezig is
  4. Stap 3: Serienummers worden automatisch gedetecteerd uit HA — vul handmatig in waar nodig
  5. Stap 4: Upload een foto van het schip
  6. Stap 5: Overzicht en opslaan
De wizard detecteert automatisch de Victron serienummers. Controleer altijd of SmartShunt en MPPT gevonden zijn — zo niet, zie sectie 9.

Mobile en kiosk dashboard aanmaken

  1. Maak een nieuw dashboard aan met naam "finally" voor mobiel gebruik
  2. Voeg een card toe via Handmatig:
type: custom:finally-skycard-mobile-customer
  1. Maak voor een vast scherm (kiosk) een tweede dashboard aan met naam "kiosk"
  2. Voeg een card toe via Handmatig:
type: custom:finally-skycard-customer
De kiosk card schaalt automatisch naar elke schermresolutie — het ontwerp is gemaakt op basis van 1920×1080 maar past zich aan op kleinere of grotere schermen zonder vervorming.

6b. Bootfoto voorbereiden

De boot van de klant wordt als losse PNG-laag over de weerachtergrond geplaatst. Voor het beste resultaat:

  • Liggend formaat — de boot breed in beeld (verhouding minimaal 1.3:1), geen portretfoto
  • Zijaanzicht van de boot, goed belicht, het hele schip in beeld
  • Transparante achtergrond — geen lucht of water, alleen de boot zelf
  • Geen lege ruimte rondom de boot — strak bijgesneden tot de rand
  • Resolutie: 1500×800 tot 2000×1000 pixels is ruim voldoende
Een portretfoto (hoger dan breed) geeft visuele problemen: de boot lijkt te zweven of valt grotendeels buiten beeld op kleinere schermen. Vraag de klant altijd om een liggende foto.

Werkwijze

  1. Vraag de klant om een foto van hun boot (zijaanzicht, het schip vult het grootste deel van het beeld)
  2. Verwijder de achtergrond met een AI-tool (bijv. ChatGPT, Remove.bg) zodat er een effen zwarte of witte achtergrond overblijft
  3. Gebruik het meegeleverde script boot_voorbewerken.py om de achtergrond automatisch transparant te maken en bij te snijden:
pip install Pillow numpy
python3 boot_voorbewerken.py invoer.png boot.png
  1. Het script waarschuwt automatisch als de verhouding niet ideaal is (te hoog/smal)
  2. Kopieer het resultaat als boot.png naar www/finally-card/ op de installatie

7. Walstroom automatisering

Er zijn 5 automations die samen het walstroom systeem beheren:

AutomationTriggerActie
SOC laag inschakelenSOC < drempel voor 2 minSocket aan (met 30 min anti-klapper)
SOC hoog uitschakelenSOC > drempel voor 5 minSocket uit
Zon uitschakelenZon > drempel voor 10 minSocket uit + timestamp opslaan
Overbelasting inschakelenLoad > 4500W voor 30 secSocket aan (altijd, ook bij override)
Overbelasting uitschakelenLoad < 2500W voor 10 min + SOC > 35%Socket uit

Drempelwaarden zijn instelbaar via het dashboard (⚙ Instellingen in de walstroom tegel):

HelperStandaardBereik
walstroom_soc_aan30%min 10, max 70, stap 5
walstroom_soc_uit80%min 50, max 100, stap 5
walstroom_zon_drempel300Wmin 0, max 1000, stap 50
De automations zijn beschikbaar als YAML bestanden. Importeer via Instellingen → Automations → drie puntjes → Bewerken als YAML.

Maak aan via Instellingen → Helpers → + Toevoegen, of importeer in één keer het bestand finally_walstroom_helpers.yaml:

9. Bekende beperkingen en openstaande punten

Viewport schaling — opgelost

  • De kiosk card schaalt nu automatisch naar elke schermresolutie via een CSS transform-schaling op een vaste 1920×1080 ontwerplaag. Getest en werkend op laptop schermen met afwijkende resolutie.

Wizard detectie

  • SmartShunt wordt gezocht via entity ID eindigend op _spanning of _state_of_charge. Bij afwijkende namen: handmatige invoer in stap 3.
  • MPPT wordt gezocht via "smartsolar" met zowel pv_power (VE.Direct) als pv_yield_power (VE.Can). Bij afwijkende namen: handmatige invoer in stap 3.
  • Handmatige invoer als automatische detectie mislukt — is al geïmplementeerd in de wizard.
  • Weather entity (weather.forecast_thuis e.d.) wordt nog niet automatisch gedetecteerd — moet momenteel handmatig in de code aangepast worden per installatie.

Wizard UX

  • Selectie in stap 2 (apparatuur) is slecht zichtbaar — kleurcontrast wordt verbeterd in volgende versie.
  • Foto upload: bij staande foto's of masten is het onderwerp soms niet zichtbaar (object-position fix volgt).

Nog te bouwen

  • Automatische generatie van finally-skycard-mobile.js op basis van wizard configuratie
  • Remote beheer via Nabu Casa (aanbevolen: €7/maand, geen open poorten nodig)
  • Multi-bank BMS ondersteuning in wizard
  • Handmatige invoer serienummers als detectie mislukt
